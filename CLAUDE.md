# Pull Request Workflow

When working on branches or pull requests in this repository, follow
@docs/ai/create-pull-request.md as the source of truth.

Key defaults:

- use `main` as the default base branch
- use branch names shaped like `<type>-<very-short-feature-name>`
- keep branch names lowercase and hyphenated
- transliterate any Tamil wording to ASCII before using it in the repo
